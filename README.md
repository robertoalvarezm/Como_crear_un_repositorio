# Instrucciones para vincular `RStudio` a `GitHub`

Este documento describe, de manera sencilla y directa, los pasos necesarios para conectar un proyecto de RStudio con un repositorio remoto en GitHub. Aquí encontrarás cómo configurar Git en tu equipo, generar tu token de acceso, crear un repositorio local y finalmente sincronizarlo con GitHub para compartir tu código o respaldarlo de forma segura.

Antes de comenzar, asegúrate de tener `Git` instalado en tu computadora. Puedes descargarlo desde [git-scm.com](https://git-scm.com/).
